 Interview Pro Simulator

 Project Overview

I created this project to make interview preparation more practical and interactive. Instead of just reading questions, I wanted to simulate a real interview environment where a user can answer questions under time pressure and evaluate their performance.

This simulator allows users to select a category (HR or Technical), attempt questions with a timer, and receive feedback based on their responses. It gives a basic idea of how well someone can perform in an actual interview setting.

---

 Key Features

* **Category Selection** – Users can choose between HR and Technical questions
* **Timed Questions** – Each question has a countdown timer to simulate real interview pressure
* **Answer Input** – Users can type answers or use voice input
* **Smart Feedback System** – Answers are evaluated based on length and relevant keywords
* **Score Tracking** – Displays performance after completing the interview
* **History Storage** – Saves previous attempts using localStorage
* **Performance Graph** – Visual representation of past scores using charts
* **Modern UI** – Clean interface with gradients, animations, and smooth transitions

---

 Technologies Used

* **HTML** – Structure of the application
* **CSS** – Styling (glassmorphism, gradients, animations)
* **JavaScript** – Core logic, timer system, scoring, and interactivity
